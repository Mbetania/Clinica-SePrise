# Clinica-SePrise

## Instalación

Sigue estos pasos para instalar el proyecto:

```bash
# Clonar el repositorio (elige una de estas opciones)
git clone https://github.com/Mbetania/Clinica-SePrise.git
# o
git clone git@github.com:Mbetania/Clinica-SePrise.git
```

Una vez clonado el repositorio:

```bash
# Instalar dependencias
pnpm install

# Construir el proyecto
pnpm run build

# Iniciar el servidor de desarrollo
pnpm run dev
```

## Acceso

Después de iniciar el servidor, accede a la aplicación desde tu navegador en:  
[http://localhost:3000](http://localhost:3000)
